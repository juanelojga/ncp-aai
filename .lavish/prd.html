🧩 User Stories

US-1 · Trigger an investigation

"Investigate reasoning and action frameworks" → full sourced study unit.

  • RAG-check before any web call
  • ≥3 distinct web sources for gaps
  • Note: Concepts · NVIDIA context · Mermaid · Terms · Refs
  • Saved < 5 min; ≥5 quiz items attached

US-2 · Review in Lavish

Highlight a section → ask follow-up → page updates live.

  • Self-contained index.html + Mermaid via CDN
  • Highlight + prompt via lavish-axi poll
  • Live-reload without losing scroll place
  • Every claim cite-able to URL or note

US-3 · Take a quiz

Exam-style MCQ with immediate feedback.

  • 4 options, one unambiguous answer
  • Rationale + citation per item
  • In-browser grading + running score
  • Answer-key audit ≥ 95% correct

US-4 · Query my knowledge base

"What do I know about Guardrails?" → grounded from my notes.

  • Top-k chunks from ChromaDB
  • Source note + heading links
  • Precision@5 ≥ 85% on held-out set

US-6 · Ingest raw documents new

Drop a PDF / Markdown / HTML → parsed, chunked, added to wiki + RAG.

  • Watched inbox/ or ingest_document(path)
  • PDF → clean Markdown (tables preserved); OCR fallback for scans
  • Provenance: file, page nos, timestamp in metadata
  • Idempotent (content-hash dedup); 100-page PDF < 60s
